Why Use LLMs?



  • Can simulate agents from a wide range of social contexts

  • Can be used as standardized interaction partners to study social phenomena

  • Can study human-AI cooperative tasks

Prompt Design



  • System Prompt
    Instructions for what persona to adopt

  • User Prompt
    Participant text input containing questions, commands, statements, etc.

  • Assistant Prompt
    Responses to User Input, constrained by System Prompt


Structured Output



  • Allows for more control over LLM output



Other Factors



  • Temperature
    Adjusts how deterministic the output will be

  • Tokenization
    How the input text is split into tokens (e.g. psychology -> psych, ology)

  • Model Versions, Rate Limits, etc.
    More complex models may require more careful designs

Pattern of Awareness
  • More updating towards desirable outcome = Higher awareness
  • More updating towards undesirable outcome = Lower awareness
